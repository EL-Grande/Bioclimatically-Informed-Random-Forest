# LeafWetness
LeafWetness: R-based Random Forest regression for leaf wetness prediction. Compares Magnitude Scaling, Min-Max, and Z-Score transformations. Evaluates loss function informativeness and metrics. Features bioclimatic models as priors to inform the ML process and uses Partial Dependence Functions (PDPs) for model interpretability.

The main script is named: Finale26.R
To run it, one just need to install the required packages invoked in the script. 
PS: Change the directory to the scripts and the data.
